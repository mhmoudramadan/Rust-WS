### Installing

Say what the step will be

1-for macOS,Linux

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

2-For windows

```
download RUSTUP-INIT.EXE for (32 or 64 Bit)

```

3- clone the repository

```
git clone https://github.com/mhmoudramadan/Rust-WS.git
```
